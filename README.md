## Overview

Alien Invasion is a Python-based arcade-style game inspired by classic space shooters. The player controls a spaceship and must defend Earth from waves of invading aliens. The goal is to survive as long as possible, eliminate enemy ships, and achieve the highest score possible.

This project was created as a final project for a Python programming course and demonstrates the core programming concepts covered throughout the class, including variables, data types, collections, control flow, loops, functions, user input, and object-oriented programming.

## Features

* Player-controlled spaceship
* Alien enemies that move across the screen
* Projectile-based combat system
* Score tracking
* Multiple waves of enemies
* Collision detection
* Game-over conditions
* Increasing difficulty as the game progresses

## Technologies Used

## Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/alien-invasion.git
```

2. Navigate to the project directory:

```bash
cd alien-invasion
```

3. Install the required dependencies:

```bash
pip install pygame
```

## Running the Game

Run the main game file:

```bash
python alien_invasion.py
```

## Controls

| Key         | Action          |
| ----------- | --------------- |
| Left Arrow  | Move ship left  |
| Right Arrow | Move ship right |
| Spacebar    | Fire weapon     |
| Q           | Quit game       |
